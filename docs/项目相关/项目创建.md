# 项目创建

## 创建react项目

1、安装react-router-dom

```
yarn add @types/react-router-dom@5.1.2 --save
```

2、安装axios

```
yarn add axios --save
```

3、安装antd

```
yarn add antd --save
```

4、安装less

```
npm i  less less-loader --save
```
